# Point Claiming System

A web-based application for managing user points with token authentication.

## Features
- Token-based authentication
- Point claiming functionality  
- User-friendly interface
- PHP backend with HTML frontend

## Files
- `claim_point.php` - Handles point claiming logic
- `fetch_token.php` - Manages API token authentication
- `index.html` - Main user interface

## Setup
1. Upload files to web server
2. Configure database if needed
3. Access via web browser
